típusai:
	- statikus NAT (SNAT) ahol a privi és pubi címek találkoznak
		elsődleges irány: befelé irány
		használata:szerverek elérhetővé tétele
		hozzárendelés típusa: kézi fix bhc -> bgc hozzárendelés (1:1)
		pub. címigény: annyi amennyi eszköz számára lehetővé szeretném tenni a publikus hálozaton keresztüli kommunikálást

		konf:
		(conf-if)#ip nat inside/outside -belső helyi háló és privát címzést használó/forgalom publikus hálóra csatlakozó int.
		ip nat |mit|mire
		
		ip nat inside source static <privi. ip cim> <cél pub. ip cím>


	- dinamikus NAT (DNAT)
		- elsődleges irány: kifelé
		- használat: kliensek/hostok esetén
		- hozzárendelés típusa: dinamikus bhc-> ? bgc 1:N
		- pub. címigény: annyi amenyi a statikus nat-nál és kevesebb is elég lehet
		
	konf.
		nat szabály:a)privát címek amiknek szabad netezni:
				access-list <sorszám 1-99> permit | deny  <IP | IP-k> 
			b)publikus címek:
				ip nat pool <név> <kezdő ip> <utolsó ip> netmask <maszk>
				ip nat inside source list <sorszám> pool <név>
